## Pattern_Generator_HDA

A SideFX Houdini HDA to generate different styles of Islamic patterns and motifs, using Shape Grammer Rules. My **research paper** and **HDA file** are included in repo. Current version of the tool was made using Houdini Apprentice (educational) so should be compatible with any liscence type.

**To use:** download the HDA file and import into your .hip file or library.
